This is where I will be updating my Modpack for Gauntlet Slayer Edition. Please read all Documentation for mods before using them.

**Important Notes:**

My mods will NOT run correctly without the mod **AddUtility**. Please be sure to have this mod on at all times, it also functions as a compatibility mod, meaning you may crash while joining other modded games without it on.

Mods are split into a few different categories-

**Clientside Mods**: These mods only work if all players are using them with identical settings. Otherwise: players without the mods and settings will likely crash when joining a Clientside modded game.

**Local Mods**: These mods only affect your own game, not others.

**Serverside Mods**: These mods affect all players as long you are the host. (These mods can have effects in them that are Local only as well)

**Trials Mods**: These mods are for Gauntlet Trials, created in the main Gauntlet Official™ Discord server (https://discord.com/invite/EkN5zA8U).
